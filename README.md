# QA Automation Exercise - C# & Selenium

**Purpose:**  
Demonstrate automation testing skills using C# and Selenium on sample web applications.

**Skills Demonstrated:**  
- C# programming fundamentals  
- Selenium WebDriver for browser automation  
- Test case design and execution  

**Tools Used:**  
- Visual Studio / VS Code  
- Selenium WebDriver  
- ChromeDriver  

**Project Details / Approach:**  
1. Created automated scripts for login, registration, and product search.  
2. Implemented basic assertions to validate functionality.  
3. Logged test results and screenshots for verification.

**How to Run:**  
- Clone the repo  
- Open project in Visual Studio  
- Ensure ChromeDriver is installed and path set  
- Run the scripts to see automation in action
